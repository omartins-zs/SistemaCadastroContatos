## Comandos 👨‍💻💻 
```bash

# Criando Migration da Tabela Contatos
$ Add-Migration CriandoTabelaContatos -Context BancoContext

# Criando BD e Fazendo Update pós da Migration
$ Update-Database -Context BancoContext

```
